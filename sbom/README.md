# Software Bill of Materials (SBOM)

`announcement-bar.cdx.json` is a [CycloneDX](https://cyclonedx.org/) 1.5 SBOM for
this package. It has **no third-party runtime dependencies** (the empty
`components` array is intentional) — at runtime it relies only on the Nioteq CMS
host. Update the `version` when releasing.
